# 🔐 SIEM Threat Detection Dashboard

Python-based SIEM that detects brute force, port scans, 
malware and failed logins — with a live Flask dashboard.

## Run it
pip3 install -r requirements.txt --break-system-packages
python3 log_generator.py
python3 app.py

## Open browser
http://127.0.0.1:5000
